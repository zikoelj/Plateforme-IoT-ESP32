# Plateforme-IoT-ESP32
Ce projet démontre la faisabilité de concevoir, simuler et déployer une plateforme IoT complète sans investissement matériel, en utilisant Wokwi pour la simulation matérielle et Node-RED pour la supervision et le contrôle. Il met en œuvre des compétences clés en IoT : communication MQTT, programmation embarquée, développement de dashboard et logging de données. Cette approche est idéale pour la formation, le prototypage rapide et la validation de concepts avant une réalisation physique.
